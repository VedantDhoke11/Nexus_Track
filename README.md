# Hackathon Management System

This is a dynamic hackathon management system connected to Supabase.

## Features
- Participant Dashboards
- Admin Management
- Judge Scoring
- Live Leaderboard (Dynamic)
- Notifications

## Tech Stack
- Next.js 14
- Tailwind CSS
- Lucide React
- Supabase (Database)
